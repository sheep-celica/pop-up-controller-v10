# Application

The normal end-user workflow for prebuilt firmware bundles is through the [Pop-up Controller V10 Application](https://github.com/sheep-celica/Pop-up-controller-V10-Application).

## What The Application Is For

- flashing released firmware bundles
- communicating with the controller over serial
- accessing controller-facing configuration and diagnostics
- reading statistical data and manufacture information

## Where To Go For App Details

App-specific instructions, releases, screenshots, and support should live primarily in the application repo:

- [Pop-up Controller V10 Application](https://github.com/sheep-celica/Pop-up-controller-V10-Application)
