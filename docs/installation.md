# Installation

This page is intentionally high-level. The detailed step-by-step installation guide is maintained outside this repo as a PDF.

## Summary

For supported Toyota Celica T18 installs, the Pop-up Controller V10 is intended to be a plug-and-play replacement for the factory Light Retractor Relay.

## What To Expect

- No custom wiring should be required for the intended plug-and-play Celica T18 installation
- Several trim pieces still need to be removed to reach the factory Light Retractor Relay location
- The exact trim-removal and access steps are described in a separate pdf install guide

## Compatibility Notes

- Direct-fit target: Toyota Celica T18 / 5th generation Celica
- Other pop-up headlight cars may be compatible with a custom adapter harness
- This repo does not currently publish wiring schematics or connector-level installation drawings
